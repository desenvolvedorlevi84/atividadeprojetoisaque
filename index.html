Conceitos do versionamento

Os comandos para alterar o usuario e email no git sao:

git config --global user.name "w3schools-test"
git config --global user.email "test@w3schools.com"

Git é um sistema de controle de versao
Github é um repositorio

O commit serve para criar um save-point das alterações do projeto

O comando push serve para enviar seu projeto para um repositorio

O comando pull serve para puxar as atualizações do projeto no repositorio
